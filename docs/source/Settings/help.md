# Help

The **Help** tab in the Settings dialog displays the current plugin version and provides quick links to external resources.

## Version Information

The top of the Help tab shows the current Geo-SAM version number, read from the plugin metadata.

## Quick Links

| Link | Destination |
|---|---|
| **Documentation** | The online Geo-SAM documentation (Read the Docs) |
| **GitHub** | The source code repository |
| **Report Bug** | Open a GitHub issue for bug reports |
| **Discussions** | Join GitHub Discussions for questions and feedback |

```{tip}
If you encounter a problem, check the {doc}`FAQ & Troubleshooting </faq>` page first — it covers the most common issues and their solutions.
```
